# سیستم مدیریت قیمت قطعات الکترونیکی

این سیستم برای مدیریت قیمت قطعات الکترونیکی طراحی شده است.

## نیازمندی‌ها

- PHP 7.4 یا بالاتر
- MySQL 5.7 یا بالاتر
- Composer
- وب سرور (Apache/Nginx)

## نصب

1. فایل‌های پروژه را در مسیر وب سرور خود قرار دهید.

2. فایل `.env.example` را به `.env` تغییر نام دهید و تنظیمات پایگاه داده را ویرایش کنید:
```env
DB_HOST=localhost
DB_NAME=your_database_name
DB_USER=your_database_user
DB_PASS=your_database_password
```

3. دستورات زیر را در ترمینال اجرا کنید:
```bash
composer install
```

4. پایگاه داده را با استفاده از فایل `database.sql` ایجاد کنید.

5. به آدرس `http://your-domain.com` مراجعه کنید.

## ویژگی‌ها

- مدیریت بردها
- مدیریت دستگاه‌ها
- مدیریت قطعات
- مدیریت کاربران
- ثبت فعالیت‌ها
- تنظیمات سیستم

## امنیت

- احراز هویت کاربران
- رمزنگاری پسوردها
- محافظت در برابر حملات SQL Injection
- محافظت در برابر حملات XSS
